Folder contains questions and solutions for "Insights from Data with BigQuery: Challenge Lab", Google Cloud Skills Boost.

The  solutions are the codes to enter into BigQuery for the respective questions/tasks.

You need to know 
* how to use BigQuery in Google Cloud Platform to query a large database.
* how to import the public dataset and table "bigquery-public-data.covid19_open_data.covid19_open_data" into BigQuery.
* how to export data from BigQuery to Google Data Studio.
* how to manipulate parameters to visualise data in Google Data Studio.

These are covered in the "Insights from Data with BogQuery" quest.

Lastly, a screengrab of my completion badge is attached to prove to you that the answers are valid for the questions/tasks as of 20 Nov 2021.
